# ngine-frontend
The frontend repository of the application. Built with Angular. For more information, check out our official documentation: https://ngine.dev/documentation.

## Get Started

1. Scaffold the frontend project with the following command: `npx ngine-frontend <your-project-name>`.

2. Install the dependencies: `npm install`.

3. Add Ngine schematics: `ng add @ngine/component` to later generate components with:
    - `ng generate base-component <name>`
    - `ng generate management-base-component <name>`.

4. Run the project: `ng serve`.
